# Presentation materials

This directory contains the editable presentation files and source brief used to develop the interactive LEGO class demo.

## Files

- `presentations/LEGO-Entertainment-Strategy-Interactive-EN.pptx` — English interactive presentation.
- `presentations/LEGO-Entertainment-Strategy-Storyline-EN.pptx` — English storyline presentation.
- `brief/source-storyline-brief.txt` — original content and narrative brief.

The website consumes its optimized images and 3D models from `src/assets/`. The files in this directory are retained for collaboration and future editing; the deployment workflow does not copy them into the public website bundle.
